# AsignacionPiezasApp (SQLite + Listado)

## Publicar
- Visual Studio: Importa `Properties/PublishProfiles/FolderProfile.pubxml` y pulsa **Publish**.
- PowerShell: `./scripts/build_publish.ps1`

## Datos
- DB: `%APPDATA%/AsignacionPiezasApp/asignacion.db`
- Fotos: `%APPDATA%/AsignacionPiezasApp/Fotos`
